# angels-website

Angel's Cleaning Services website for Comox Valley, BC.

## How to Launch

This is a static HTML website. To launch it locally:

1. Using Python (recommended for testing):
   ```bash
   python3 -m http.server 8080
   ```
   Then visit http://localhost:8080 in your browser.

2. For production deployment, simply upload all files to any static web hosting service (GitHub Pages, Netlify, Vercel, etc.)

## Structure

- `index.html` - Main homepage
- `Contact-us.html` - Contact page
- `commercial.html` - Commercial cleaning services
- `residential.html` - Residential cleaning services
- `airbnb.html` - Airbnb cleaning services
- `images/` - All image assets
- `logo.png` - Company logo

## Notes

- The website uses inline CSS and JavaScript (no external dependencies)
- All styling is self-contained within each HTML file
- `script.js` and `style.css` are not currently used by the website
